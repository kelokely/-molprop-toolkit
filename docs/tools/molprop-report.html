molprop-report

Builds a shareable report bundle from a results CSV. The report includes category summaries and plots and writes both Markdown and HTML output into a timestamped directory under reports/.

calc-v4 calc-v5 prep analyze report sketch series picklists similarity featurize retro schema

Typical usage

molprop-report results.csv

What you get

A folder like reports/<stem>_report_YYYYMMDD_HHMMSS/ containing report.md, report.html, and a plots/ subfolder.

Tip

If you want a clean, portable artifact for collaborators, send the entire report directory (or zip it). The HTML report is intended to be readable without a Python environment.
