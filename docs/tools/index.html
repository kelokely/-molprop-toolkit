CLI tool reference

Each page below documents a single CLI command with common options and examples. For the authoritative, always-up-to-date help, run <command> --help.

Calculators

molprop-calc-v4

Baseline medchem property table from SMILES (CNS MPO, alerts, oral filters, etc.).

molprop-calc-v5

Extended table with solubility/permeability/PK + optional 3D shape descriptors.

Ligand preparation (guide)

Recommended standardization sequences: salts/charges → tautomers → stereo → protomers → 3D.

Tip

If you plan to use 3D descriptors or USR/USRCAT clustering, decide stereochemistry and protonation state up front. The toolkit records those choices in Stereo_* and Protomer_* columns when enabled.

Analysis & reporting

molprop-analyze

Master analyzer: list categories, preview a category, or generate a text report.

molprop-report

Builds a Markdown + HTML report bundle with plots.

molprop-sketch

2D structure depictions from a results CSV, with optional HTML index.

molprop-series

Series analytics: Murcko scaffolds + similarity clustering.

Operational triage

molprop-picklists

Generate decision picklists (CSV and optional HTML) from filters/sorts.

molprop-retro

Offline retrosynthesis wrapper using AiZynthFinder (summary CSV/JSON + HTML route browser).

validate schema (utility)

Validate that a results CSV matches the toolkit schema and required categories.

Tip

Retrosynthesis is optional. If you do not install AiZynthFinder, the rest of MolProp Toolkit works as usual.
